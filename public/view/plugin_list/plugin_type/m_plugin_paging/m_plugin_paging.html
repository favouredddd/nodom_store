{{name}}插件

因分页插件应用场景的不同，当当前页发生变化时，需要请求数据，因此需要在插件模块中定义updatePage函数，具体用法详见NoDom官网模块中方法的使用


HTML代码

                                        
<div class="plugin-paging" x-plugin="paging_big"></div>

JS代码

                                    
{

                                    
      name: 'paging',      /*插件模块的模块名称*/

                                    
      el: '.plugin-paging',      /*插件渲染容器*/

                                    
      data: {      /*配置参数项*/

                                    
            page: 1,      /*分页插件当前页*/

                                    
            row: 10,      /*分页插件每页条数*/

                                    
            total: 30,      /*分页插件总条数*/

                                    
            to_page: 1      /*分页插件需跳转页数*/

                                    
            page_color: '#FF0000'      /*分页插件页数颜色*/

                                    
     }

                                    
}

插件说明:


本分页插件不仅有普通插件的翻页功能，点击即跳转至目的页码，并可以直接跳转到首页或末页。更额外添加了中间页码的智能展示、记录总数展示以及总页数展示。本插件可自定义配置项包括：页数颜色（总页数、总条数、当前页）、当前页、每页条数、跳转页数、总条数。


HTML代码

                                        
<div class="plugin-paging" x-plugin="paging_big" dataName="paging_data"></div>

JS代码

                                    
{

                                    
      name: 'paging',      /*插件模块的模块名称*/

                                    
      el: '.plugin-paging',      /*插件渲染容器*/

                                    
      data: {      /*配置参数项*/

                                    
            paging_data: {      /*分页插件当前页*/

                                    
                  pre_page: 1,      /*分页插件当前页*/

                                    
                  go_page: 1,      /*分页插件需跳转页数*/

                                    
                  all_page: 16,      /*分页插件总页数*/

                                    
                  page_rows: [],      /*插件渲染需要，只需传入空数组*/

                                    
                  pre_color: '#5eaee3',      /*当前页颜色*/

                                    
                  word_color: '#999999'      /*页码字体颜色*/

                                    
            }

                                    
     }

                                    
}

插件说明:


分页插件不仅有普通插件的翻页功能，更额外添加了中间页码的智能展示，点击即跳转至目的页码。分页插件需配置当前页、跳转页、总页数。
