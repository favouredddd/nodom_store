     }

插件说明:

                                    

                                    
这是一款模仿苹果开关的switcher插件，通过点击进行开关变换。使用插件时需设置插件渲染容器父元素的宽度、高度，switcher插件根据父元素宽度、高度自适应设置自己的高度、宽度，本插件可自定义配置项包括：打开颜色、关闭颜色、按钮颜色。
