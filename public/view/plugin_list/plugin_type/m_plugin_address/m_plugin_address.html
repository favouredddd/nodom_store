{{name}}插件

本章节主要介绍插件库选择地址插件，主要用于定位，选择所属省份等。


HTML代码

                                        
<div class="position">

                                        
      <div class="location" x-plugin="address"></div>

	                                    
</div>

JS代码

                                    
{

                                    
      name: 'address',      /*插件模块的模块名称*/

                                    
      el: '.location',      /*插件渲染容器*/

                                    
      data: {

                                    
            location_country:  '*****'      /*当前位置城市名*/

                                    
            popular_country: [{name: '北京'}]      /*热门城市数组*/

                                    
            small_div: {      /*颜色对象*/

                                    
                  color_1: '#ffffff',      /*热门城市bg颜色*/

                                    
                  color_2: '#66d9ef',      /*定位颜色*/

                                    
                  color_3: '#457eb1',      /*城市列表标题颜色*/

                                    
                  color_4: '#5a8dba'      /*热门城市边框颜色*/

                                    
      }

                                    
}

插件说明:


这是一款选择与获取当前省份的智能Location插件。插件分为"当前省份"、"热门省份"以及按照拼音首字母排列的各个省份三个部分。通过点击省份名称，来改变"当前省份"。"
