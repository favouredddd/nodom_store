     }

插件说明:


这是一款图片放大镜插件。可以一定程度上放大鼠标选中的区域。本插件可自定义配置项包括：放大比例、颜色、透明度、图片路径。在使用时，需定义图片标签的父元素标签的大小（width,height）。
